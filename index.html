Kickoff

A lightweight front-end framework
for creating scalable, responsive sites

Download Kickoff now

To get started, simply checkout the framework using Git:

git clone https://github.com/tmwagency/kickoff.git your-project-folder

..or with the Github desktop app on Mac or Windows

Kickoff is built with:

  • Grunt
